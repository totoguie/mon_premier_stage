Description du poste

Nous recherchons un(e) stagiaire développeur web junior pour rejoindre notre équipe de développement. Vous participerez à la création d'applications web modernes en utilisant les dernières technologies.

Ce stage vous permettra d'acquérir une expérience pratique dans le développement d'applications web complètes, de la conception à la mise en production. Vous travaillerez en étroite collaboration avec notre équipe expérimentée sur des projets réels pour nos clients.

Missions principales :

  • Développement d'interfaces utilisateur avec React.js
  • Création d'APIs REST avec Django
  • Intégration de bases de données PostgreSQL
  • Tests unitaires et d'intégration
  • Participation aux revues de code
  • Documentation technique

Compétences requises

Compétences techniques :

HTML/CSS JavaScript React.js Python Django Git SQL

Qualités personnelles :

  • Curiosité et passion pour les nouvelles technologies
  • Capacité à travailler en équipe
  • Autonomie et prise d'initiative
  • Rigueur et attention aux détails
  • Bonne communication écrite et orale

Formation :

Diplôme en Informatique, Génie Logiciel ou formation équivalente. Niveau Bac+2/Bac+3 minimum.
